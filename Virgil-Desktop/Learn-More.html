Virgil: A Simple Mobile UI

What is Virgil?

Virgil makes your phone easier.
It's a simplified User Interface with customizeable options that allows you to display the features you want on your smartphone display. No more smartphone sensory overload. We've designed Virgil for non-smartphone people.
Whether your user is a child, elderly, special needs, or just plain technologically challenged, Virgil can help make you and your' families lives simpler, easier and safer.

Simple

Core features from your phone simplified into a clean, large, easy to read user interface. No more fumbling with tiny keyboards. Everything you need is directly accessible.

Essential

No more smartphone overload: Virgil lets you customize clear, concise options people can understand, and gives you only the essential features you need.

Universal

Virgil works with all major smartphones. To fully utilize all features, we recommend a phone with the latest web browsers and geolocation.
